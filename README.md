# CJSI-react
Hometask for Ciklum JS Internship Lesson 7

---

##### List of used technologies:
- React.js Library
- webpack

---

##### To run the project locally, use:
```
git clone https://github.com/AlexMarvelo/CJSI-react.git
cd CJSI-react
npm i
npm run start
```
Then open *http://localhost:3000/* in your browser
